The core module provides the fundamental components of LangLint.

Dispatcher

Parser dispatcher for routing files to appropriate parsers.

This module provides the core dispatcher that determines which parser should handle a given file based on file extension, MIME type, and content.

Caching system for LangLint.

This module provides caching functionality for parse results and translations to improve performance and reduce API calls.
